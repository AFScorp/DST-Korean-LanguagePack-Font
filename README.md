# DST-Korean-Localization-Pack
Don't Starve Togetehr Korean Language Pack
굶지마 다함께 한글화 모드

릴리즈할 때는 클라이언트와 서버 버전을 별도로 하여, modmain 폴더의 내용물에 각각 client 폴더와 server 폴더의 modinfo.lua와 합친 후 모드 업로더로 릴리즈하면 됩니다.
